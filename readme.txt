Simple Smart Offers upsells every product except what is currently in the cart

== Description ==

Simple Smart Offers upsells every WooCommerce product on your site except what is currently in the cart.

== Installation ==

1. Upload this folder to the `/wp-content/plugins/` directory or install from the Plugins menu in WordPress
1. Activate the plugin through the Plugins menu in WordPress
1. Go to the “Simple Smart Offers” tap in the WP backend and add some smart offers to be shown during checkout
